# CWOps Practice MP3 Generator

## Requirement

- Node >= 13.x
- ffmpeg installed

## Installation

- `git clone git@github.com:mdp/cwops_practice.git`
- `cd cwops_practice`
- `npm install`
- `npm run generate`
- Wait, this will take a while

## Customization

Take a look at cwops_beginner.json for options
